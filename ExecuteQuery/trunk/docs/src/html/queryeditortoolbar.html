Query Editor Toolbar
The following commands are available from the editor's toolbar
Description Shortcut
Executes ALL statements within the editor F5
Executes the statement at the current cursor position (statements must be separated by at least one new-line character) Ctrl+ENTER
Executes any selected text within the editor Ctrl+SHIFT+ENTER
Cancels the currently executing statement
Clears and resets the output pane
Displays the executed statement history dialog Ctrl+H
Select the previously executed statement
Select the next executed statement
Commits all changes for the last transaction block Ctrl+SHIFT+C
Rolls back all changes for the last transaction block Ctrl+SHIFT+R
Toggles the auto-commit mode on or off
Recycles the editor's current connection and retrieves a new one from the pool
Displays the selected result set's meta data
Exports the selected result set to file
Shows/hides the query statement output pane
Shift the current line/selection left Ctrl+D
Shift the current line/selection right Ctrl+T
Comments the current line/selection Ctrl+SHIFT+D
Un-comments the current line/selection Ctrl+SHIFT+C
