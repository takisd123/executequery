Exporting Result Sets

Result sets of executed queries may be exported to delimited file or Excel spreadsheet format as outlined below.
  1. Select the result set to be exported from the editor's output pane
  2. Select the Export the selected result set to file button from the editor's toolbar to display the export dialog
  3. Select the relevant options depending on the export type required
  4. Select a valid output file path
  5. Select the dialog's OK button to begin the export process
  6. A small progress dialog will appear followed by notification of the process completing successfully or otherwise
