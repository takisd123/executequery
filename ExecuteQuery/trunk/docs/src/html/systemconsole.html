System Console Docked Component

The System Console component provides a simple local system console for use from within Execute Query. This does not aim to replace the resident system terminal within any particular operating system but rather to provide a quick and simple means of viewing system terminal output from within the application as desired.

The System Console may be opened using the menu item Tools > System Console Window or the from the System Console button from the main tool bar.

The default location for the console is as a docked tab component within the center-bottom area of the application.
