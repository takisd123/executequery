User Defined SQL Key Words
Adding key words to the application allows for key word formatting to be applied to non-SQL92 key words within relevant SQL text display components such as the Query Editor and the numerous SQL object creation features.
  1. Select the menu item Tools > User Defined Keywords or the User Defined Keywords button from the main tool bar
  2. The left-hand table display lists those key words previously added, the right-hand display lists the standard SQL92 key words
  3. Enter the new key word within the Add Keyword text field
  4. Select the Add button to include the new key word within the list
  5. Remove a user defined key word by selecting that key word from the left-hand user defined list and select the Delete (trash icon) button
  6. Select the Save button to store your changes and to reflect any modifications within any components displaying formatted and stylised SQL text
