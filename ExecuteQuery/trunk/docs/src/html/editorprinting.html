Printing Queries and Result Sets

SQL query text and generated result set tables may be individually printed using the standard printing controls available.

Printing SQL Queries from the Editor
  1. Select the editor within the central pane where the printing is to be retrieved from
  2. Select the menu item File > Print or the Print button from the application's main tool bar
  3. Select the print area from the selection dialog as the SQL Query Text Area
  4. Select the dialog's OK button to display the standard print dialog
  5. Print

Printing Result Sets
  1. Select the editor within the central pane where the printing is to be retrieved from
  2. Select the result set to be printed from the editor's output pane
  3. Select the menu item File > Print or the Print button from the application's main tool bar
  4. Select the print area from the selection dialog as the SQL Table Results Panel
  5. Select the dialog's OK button to display the standard print dialog
  6. Print
