Saving and Exporting ERDs

Generated ERDs may be saved for later editing or exported to selected image formats. Default ERD file saving occurs using simple Java object serialisation with the file extension 'eqd'. Saved ERDs may be reopened later using the File Open command from the main menu or toolbar by selecting the desired eqd file and selecting the New ERD window from the File Open dialog.

The process for saving and exporting is outlined below.
  1. From an open ERD central pane select the main menu item File > Save or select the Save button from the main toolbar
  2. Select the file or image format to save to
    Available image formats include:
    1. JPEG
    2. GIF
    3. PNG
    4. SVG (uses Apache's Batik SVG Toolkit)
    5. Select any image format specific options
    6. Select a valid file path to save to
    7. Select the Save button
