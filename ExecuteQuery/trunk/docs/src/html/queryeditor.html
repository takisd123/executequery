The Query Editor

The Query Editor is a configurable SQL statement viewing and execution tool displayed within the central pane. Any number of editors may be open at any one time.

The editor supports the following features:

  • Customisable SQL syntax highlighting
  • Bracket matching
  • Multiple query execution
  • Multiple query result set execution and display
  • Complete printing support
  • Transaction control
  • IDE style text editor functions - search, replace, insert etc
  • Result set export
  • Multiple open connection support
  • Searchable executed SQL statement history
