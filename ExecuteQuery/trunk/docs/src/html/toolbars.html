Toolbars

Execute Query includes 7 main toolbars at the top of the application. Each toolbar may be customised as required through the user application preferences that allow for the reordering and visibility of each button.

Toolbars may be moved horizontally and vertically or resized to suit using each toolbar's move widget on left-most edge. Click within the widget area and drag as desired.

Toolbars may be made visible or not either using the selection menu items at View > Toolbars > <toolbar-name> or via the user application preferences.

A description of each toolbar and its associated buttons and actions follows. Where a shortcut is not available or desired for any particular function, these may be configured within the user application preferences.
