Importing and Exporting Data

The Import/Export tools provide for the importing and exporting of data using delimited files, XML and Excel spreadsheet formats.

Please consult the relevant import and export help documents for specific information on each:
  • Exporting Data:
    • Exporting to delimited file
    • Exporting to XML file
    • Exporting to Excel spreadsheet
  • Importing Data:
    • Importing from delimited file
    • Importing from XML file
  • Date pattern formats for import of date/time column values

All import and export actions are logged to the relevant file for later reference. Refer to System Logging for further information.
