Generating Result Set Metadata

The Query Editor provides for the generation of result set meta for previously executed queries where the resulting output was a result set type.

Note: This feature is not available by default and must be enabled from the user application preferences .
  1. Select the result set to be queried from the editor's output pane
  2. Select the Display this result set's meta button from the editor's toolbar (this button will be disabled if the result meta data option has not been enabled from the user application preferences
  3. A new result set tab shall be inserted to the right of the queried result set displaying that result set's meta data
