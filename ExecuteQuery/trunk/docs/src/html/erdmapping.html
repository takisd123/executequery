Mapping an Existing Database

Existing databases from an open connection may be reverse-engineered to generate the ERD using the menu item Database > Generate ERD > Map Existing Database as outlined below.
  1. Select menu item Database > Generate ERD > Map Existing Database
  2. Select an open connection
  3. Select the schema containing the tables to be generated
  4. Select the tables to include within the ERD using the arrow selection buttons
  5. Select the Generate button

The resulting ERD may be customised to suit using the functions available as outlined in Manipulating Tables.
